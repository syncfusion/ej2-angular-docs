# Migration from Essential JS 1

This article describes the API migration process of the Spreadsheet component from Essential JS 1 to Essential JS 2.

| Save the edited cell value | **Method:** *XLEdit.saveCell* <br><br>`<ej-spreadsheet id="spreadsheet">`<br>`</ej-spreadsheet>`<br>**TS**<br>`ngAfterViewInit(){ let xlObj = $("#spreadsheet").data("ejSpreadsheet");  xlObj.XLEdit.saveCell(); }`| **Method:** *endEdit* <br><br>`<ejs-spreadsheet #default>`<br>`</ejs-spreadsheet>`<br>**TS**<br>`@ViewChild('default') public spreadsheetObj: SpreadsheetComponent;`<br>`this.spreadsheetObj.endEdit();`|
